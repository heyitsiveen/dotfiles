## Setting up New Macbook Air M2
1. [CleanShot X](https://cleanshot.com/) - **Paid ($29/$10 Monthly)**
2. [Final Cut Pro](https://www.apple.com/final-cut-pro/) - **Paid ($299.99)**
3. [Zen Browser](https://zen-browser.app/download/)
4. Clean Up **Dock** 
   - Alternative Free: [DaVinci Resolve](https://www.blackmagicdesign.com/products/davinciresolve)
5. Update **macOS**
6. Basic **Settings** Setup
   - **Settings**
     - **Network**
       - Firewall - `Enabled`
     - **Accessibility**
       - **Motor**
         - **Pointer Control**
           - Trackpad Options > Dragging style - `Three Finger Drag`
      - **Menu Bar**
        - Battery > Battery Options > Show Percentage - `Enabled`
      - **Desktop & Dock**
        - **Dock**
          - Minimize windows into application icon - `Enabled`
          - Automatically hide and show the Dock - `Enabled`
          - Show suggested and recent apps in Dock - `Disabled`
        - **Mission Control**
          - Group windows by application - `Enabled`
       - **Displays**
         - **Advanced**
           - Show resolutions as list - `Enabled`
       - **Trackpad**
         - **Point & Click**
           - Tracking speed - `Fast`
           - Secondary click - `Click or Tap with Two Fingers`
           - Tap to click - `Enabled`
       - **Scroll & Zoom**
         - Natural scrolling - `Disabled`
       - **More Gestures**
         - Mission Control - `Swipe Up with Four Fingers`
   - **Finder**
     - **Settings**
       - **General**
         - Show these items on the desktop - `Unchecked All`
         - New Finder windows show - `Downloads`
       - **Advanced**
         - When performing a search - `Search the Current Folder`
      - **View**
        - Show Path Bar
        - Show Status Bar
        - **Customize Toolbar**
          - `New Folder` - `Delete` - `AirDrop` - `Action` | `View` | `Group` | `Search`
7. [Homebrew](https://brew.sh/)
8. [Keycastr](https://github.com/keycastr/keycastr)
   - Install using homebrew: `brew install keycastr --cask`
9. [Cursor Pro](https://appahead.studio/apps/cursor-pro/) - **Paid ($14.99)**
10. [Git](https://git-scm.com/install/mac)
    - Install using homebrew: `brew install git`
